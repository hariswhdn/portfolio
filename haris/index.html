Animated Comics

Product that can compete in the market, for creators and readers satisfication.

My role

Turn research and ideas to designing interface and implement to a functional application.

Project goal

People want to see something more exciting than comics, while making animation is not that easy. Animated Comics come to simplifies animation creation through more exciting comics.

Target audience

Anyone who have hobby to read comics and watch movies.

Key challenges

SVG file format animation tools is not many available for now.

Research study

An alternative for comic and animation creators in providing content for comic and animaton fans.

User testing
Usability study insights

Based on the theme that

1. Do not understand the selection of favorite categories.

2. Don't understand how to make money by submitting works.

3. Confused to find a question and answer page.

4. Confused with the word "merchandise".

5. Confuse to check verification code from email.

An insight is

1. Define categories based on what the user has read.

2. Create a more complete creator profile page so that readers understand how the content is created.

3. Add a faq menu in the sidebar.

4. Make questions and answers on the faq menu.

5. Added link to popup how to verify email.
